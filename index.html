Introduction

Welcome to the 18th Street Cafe System! This comprehensive guide will help you install, configure, and utilize the system to streamline your cafe operations. Follow these instructions carefully for optimal performance and functionality.

System Requirements

  • XAMPP (Version 8.0 or higher) for PHP, MySQL, and Apache server
  • Visual Studio Code (Latest version recommended) for code management
  • Modern Web Browser (Chrome, Firefox, Edge, or Safari)
  • Minimum 4GB RAM, 10GB free disk space

Installation Process

Step 1: Essential Software Setup

  1. Install XAMPP
    • Download the latest version from www.apachefriends.org
    • Run the installer with default settings
    • Verify installation by launching XAMPP Control Panel
  2. Install Visual Studio Code
    • Download from code.visualstudio.com
    • Install recommended extensions for PHP development:
      • PHP Intelephense
      • PHP Debug

Step 2: Database Configuration

  1. Launch Services
    • Open XAMPP Control Panel
    • Start Apache and MySQL services (ensure ports 80 and 3306 are available)
    If services fail to start, check if other applications are using the required ports.
  2. Create Database
    • Navigate to http://localhost/phpmyadmin/
    • Click on "Databases" tab
    • Create a new database named 18thstreet with UTF-8 Unicode collation
  3. Import Database Structure
    • Select the newly created 18thstreet
    • Click "Import" tab
    • Browse and select the provided 18thstreet.sql file
    • Click "Go" to execute the import
    • Confirm all tables appear correctly (users, menu, orders, inventory, payments)

Step 3: Project Configuration

  1. Deploy Project Files
    • Extract the 18th_street_cafe folder to C:\xampp\htdocs\ (Windows) or /Applications/XAMPP/htdocs/ (Mac)
    • Ensure all files maintain their original directory structure
  2. Configure Database Connection
    • Open the project in Visual Studio Code
    • Locate and edit db/database.php in the root directory:
    <?php
// Database connection parameters
$host = "localhost";
$user = "root";
$pass = "";
$db = "18thstreet";

// Establish connection
$conn = new mysqli($host, $user, $pass, $db);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

// Set character encoding
$conn->set_charset("utf8mb4");
?>
  3. Verify File Permissions
    • Ensure the uploads and temp folders have write permissions
    • For Linux/Mac: chmod 755 for directories and chmod 644 for files

Step 4: System Launch

  1. Verify Services
    • Confirm both Apache and MySQL are running in XAMPP Control Panel
    • Check for any error messages in the logs
  2. Access The Application
    • Open your preferred browser
    • Navigate to http://localhost/18th_street_cafe/
    • You should see the login screen
  3. Initial Login
    • Use the default administrator credentials:
      • Username: admin
      • Password: admin123
    Important: Change the default password immediately after first login for security purposes.
  4. Administrator Access
    • To directly access the administrator panel, navigate to: http://localhost/18th_street_cafe/admin/admin_login.php
    • Use the same administrator credentials mentioned above

Troubleshooting Guide

Common Issues and Solutions

Issue Possible Cause Solution
Database connection failure MySQL service not running Start MySQL in XAMPP Control Panel
Incorrect database credentials Verify settings in database.php
Page not found (404) Incorrect URL path Ensure correct folder name in URL
Missing index file Check if index.php exists in root directory
Image upload failure Permission issues Check folder permissions for uploads directory
File size too large Modify PHP settings in php.ini
Slow performance Database needs optimization Run provided maintenance script optimize_db.php
High server load Consider upgrading server resources
Error Logging

The system logs errors to logs/system_errors.log. Check this file when troubleshooting to identify specific issues.

Maintenance Recommendations

Database Backup

Back up the database weekly using PHPMyAdmin export function

System Updates

Update the system when new versions are released

Performance Optimization

Run the optimization script monthly: http://localhost/18th_street_cafe/admin/optimize.php

Security Updates

Check for security updates to XAMPP and PHP regularly
